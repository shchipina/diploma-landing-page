# Інструкція з генерації документації / Documentation Generation Guide

Для автоматичної генерації документації з коду використовується **TypeDoc** 
1. Переконайтеся, що всі залежності встановлені: `npm install`
2. Запустіть скрипт: `npm run docs:generate`
3. Результат буде збережено у папці `docs-build/`. Відкрийте `docs-build/index.html` у браузері.
