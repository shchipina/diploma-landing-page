# Платформа для обміну навичками — Лендінг бакалаврської роботи

Цей репозиторій містить вихідний код односторінкового веб-сайту (лендінгу), розробленого для презентації бакалаврської роботи на тему: **«Розробка веб-платформи для обміну навичками та знаннями (Medium clone)»**.

Проєкт розроблено в рамках лабораторної роботи з фокусом на сучасні веб-технології, семантичну верстку, доступність, SEO-оптимізацію та використання системи контролю версій Git

## Технологічний стек
* **Фреймворк:** React (з Vite)
* **Мова:** TypeScript
* **Стилізація:** Tailwind CSS
* **Локалізація:** `react-i18next` (підтримка Української та Англійської мов)
* **Дизайн-патерн:** Натхненний SaaS-лендінгами (стиль Unbounce) з використанням CSS-мокапів браузера.

## Стратегія гілкування (Git Strategy)
У цьому проєкті використовується стратегія **Release Flow**:
1. Основна розробка ведеться в ізольованих гілках `feature/*`.
2. Після завершення етапу фічі зливаються в стабільну гілку `main`.
3. Для фіксації готових версій продукту створюються гілки релізів (`release/v1.0.0`, `release/v2.0.0`).
4. Багфікси на продакшені (hotfixes) спочатку вносяться в `main`, а потім переносяться в гілку релізу за допомогою `git cherry-pick` (правило "Upstream First").

## Стандарти документування
Всі нові функції, компоненти та хуки **повинні** бути задокументовані у форматі **TSDoc**
* Обов'язково використовуйте теги `@description`, `@returns` та `@example`
* Для опису складної бізнес-логіки використовуйте тег `@architecture`
* Перевірка наявності документації автоматизована через ESLint (`eslint-plugin-jsdoc`). Коміт буде відхилено системою CI, якщо публічний інтерфейс не описаний
